Десктопное приложение на Python (Tkinter) для генерации случайных цитат с возможностью:
- Генерация случайной цитаты из библиотеки
- Добавление собственных цитат (текст, автор, тема)
- Просмотр истории всех сгенерированных цитат
- Фильтрация истории по автору и/или теме
- Сохранение и загрузка данных в формате JSON
- Валидация ввода при добавлении новых цитат

## Как запустить

### Требования
- Python 3.8+
- Стандартная библиотека (Tkinter, json, random — всё встроенное)

### Запуск
1. Скачайте файл `quote_generator.py`
2. Откройте терминал в папке с файлом
3. Выполните:
   ```bash
   python quote_generator.py
